Questions fréquement posées
  • J'ai oublié le mot de passe de mon iPhone

    Le contenu de votre appareil Ios s'effacera si vous tentez d'inscrire un mauvais mot de passe à 4 reprises. Avant d'atteindre les 4 essais erronés et de perdre vos données, communiquez avec le Centre de Service TIC.

  • Je veux réinitialiser mon code de déverrouillage IPhone ou mon appareil est verrouillé et je ne me souviens plus de mon mot de passe

    Si vous voulez réinitialiser votre code de verrouillage IPhone avant son expiration :

    1. Aller sous Réglages / Codes et inscrire votre code actuel
    2. Aller ensuite sur Changer le code et suivre les instructions à l’écran

    Si votre appareil est verrouillé et que vous ne vous souvenez plus de votre code de déverrouillage, le contenu de votre appareil iOS s'effacera si vous tentez d'inscrire un mauvais mot de passe à 4 reprises. Avant d'atteindre les 4 essais erronés et de perdre vos données, communiquez avec le Centre de services des TIC pour que votre mot de passe soit réinitialisé.

  • Je ne reçois pas mes courriels sur mon iPhone alors que ça fonctionne dans Outlook

    Plusieurs causes peuvent empêcher la synchronisation des courriels entre un appareil mobile et le serveur de messagerie. Voici quelques suggestions à essayer selon le cas.

    Note : Si votre mot de passe BUR a expiré sur votre ordinateur, vous ne pourrez plus vous connecter à partir de votre appareil mobile. Vous allez recevoir, sur votre ordinateur, un message 5 jours avant l'expiration de votre mot de passe.

    1. Vérifiez si vous avez accès à Internet en accédant à un site Web avec Safari sur votre appareil. Si ce n'est pas le cas:
      • Vérifiez si le signal cellulaire ou Wifi de votre appareil réseau est bon. (En haut à gauche de votre appareil)
      • Redémarrez complètement votre appareil en appuyant simultanément sur le bouton d'allumage et sur le bouton « Home » pendant 10 secondes. Rallumez-le en appuyant sur le bouton d'allumage.
    2. Allez dans le menu Réglages de votre appareil, puis dans Mail, Contacts, Calendriers. Appuyez sur Exchange HQ et sur Compte. Entrez votre mot de passe et appuyer sur Ok. Normalement, il devrait avoir des crochets bleus si les informations sont correctes.
    3. Vérifiez si une règle de transfert des courriels est utilisée dans Outlook.
      • Désactivez les règles dans Outlook.
      • Menu Outils > Règles et alertes ...
      • Décochez les règles, puis cliquer sur OK.

      Si toutes ces démarches s'avèrent infructueuses, communiquez avec le Centre de services des TIC.

    • Certains courriels donnent une erreur lors de l'envoi

      Ce problème peut arriver lorsqu'on répond à un ou des destinataires possédant un accent (ex. é, è, ê, ö) dans leur nom. Ceci est causé par la version actuelle du serveur Exchange avec les appareils mobiles.

      Une mise à jour des infrastructures de messagerie est prévue en 2014, ce qui corrigera le problème.

      En attendant, vous devez supprimer le nom de famille du contact et conserver le prénom.

      Pour ce faire, il vous suffit d'appuyer sur Répondre ou Répondre à tous à un courriel, puis, d'appuyer sur le contact en problème puis sur la touche Effacer.

    • J’éprouve des difficultés avec la synchronisation de mon calendrier

      Plusieurs problématiques peuvent survenir, par exemple :

      • Des convocations s'affichent en doubles dans le calendrier Outlook
      • Les convocations planifiées et/ou acceptées diffèrent entre le calendrier de l'appareil mobile et le calendrier Outlook

      Voici certaines recommandations à suivre pour assurer un bon fonctionnement de la synchronisation du calendrier :

      • Recommandations - Problèmes de calendrier [pdf - 256Ko] 

      Note : La resynchronisation complète du calendrier peut régler certaines des problématiques énumérées ci-haut.

    • Je veux sélectionner un calendrier par défaut

      Si vous avez plusieurs comptes de courrier configurés sur votre appareil, assurez-vous que le compte Exchange HQ est celui par défaut pour éviter que vos rendez-vous soient créés dans le mauvais calendrier. Pour ce faire, suivez les indications suivantes :

      1. Sur votre appareil, aller dans les Réglages.
      2. Aller dans Mail, Contacts, Calendrier.
      3. Dans les sections Mail, Contact et Calendrier, vérifier les paramètres Compte par défaut pour qu'ils soient à Exchange HQ.
    • Je veux mettre à jour le logiciel de mon appareil

      Note : Veuillez attendre les directives du groupe - Technologie AVANT d'effectuer la mise à jour du système d'exploitation d'Apple.

      Pour mettre à jour votre appareil, vous devez aller dans le menu Réglages et dans Général. Appuyez sur Mise à jour logiciel. La dernière version disponible vous sera proposée. Appuyez sur Télécharger et Installer.

      Note : Vous devez être connectés à un réseau Wifi pour effectuer le téléchargement et l'appareil doit avoir au minimum 50 % de charge pour la batterie.

    • Je n’ai pas de iPhone, mais j’aimerais m’en procurer un. Comment faire?

      Seuls les cadres peuvent se prévaloir d'un téléphone intelligent. Les téléphones intelligents de type iOS (iPhone) sont disponibles lors d'un bris d'équipement ou de la nomination d'un nouveau gestionnaire seulement.

      Pour commander un appareil mobile ou intelligent de même que des accessoires, consultez le portail TELUS réservé aux employés d'Hydro-Québec.
